HOW TO PLAY

          

            
Sprint — 表示された実コードのスニペットを1文字ずつ正確に。正しい文字だけが進み、Enter で改行（先頭インデントは自動スキップ）。クリアで WPM・正確率・ランクと弱点分析。

            
Falling Tokens — => や .map( などのトークンが落下。先頭文字を打つとロックオン、最後まで打って撃破。床に落とすとライフ減少、0 でゲームオーバー。

            
Code Runner — コードの行を打ち続けてトンネルを疾走。正打で加速・ミスで減速し、ミスが続くとクラッシュ。走行距離がスコア。

            
Esc ポーズ · Tab やり直し · Backspace 戻る · 画面下の 🔊 でミュート
